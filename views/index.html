Url Shortener

Creates and stores a shortened link to the inputed URL.

Shortened Url Creation

https://ssc-url-shortener.glitch.me/ + {url you want shortened}

Creation Example Input

https://ssc-url-shortener.glitch.me/https://www.google.com

Creation Example Output

{original : "https://www.google.com", shortened : 0}

Usage

https://ssc-url-shortener.glitch.me/0

now redirects to...

https://www.google.com
